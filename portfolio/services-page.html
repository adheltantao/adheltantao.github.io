CSS Grid architecture

Services grid, process steps, and testimonials all use CSS Grid with auto-fit and minmax - no media query breakpoint juggling needed. Each section reflows naturally to any screen width.

Section structure

Nav, Hero, Stats, Services, Process, Testimonials, Contact, Footer - this is a full marketing page architecture. Each section has a clear job: convert, build trust, reduce objection, close.

Sticky nav

Navigation uses position: sticky with backdrop-filter blur for a glass-morphism effect on scroll. The CTA button in the nav stays visible at all times - a conversion best practice.

Contact form

Two-column contact section: left side builds trust with direct contact details, right side lowers friction with an inline form. Both are visible without scrolling on desktop.
